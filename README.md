## Файлы и переменные окружения

## Основные команды

### Установка

```bash
npm install
```

### Запуск dev-сервера с отслеживанием изменений — [http://localhost:8080](http://localhost:8080)

```bash
npm run dev
```

### Сборка development

```bash
npm run build:dev
```

### Сборка production

```bash
npm run build:prod
```

### Запуск web сервера для development/production сборки — [http://localhost:8080](http://localhost:8080)

```bash
npm run serve
```

### Запуск линтеров (JS)

```bash
npm run lint
npm run fix
npm run prettier
```
